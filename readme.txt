你需要准备好的东西：  
  
1. 安装 busybox
2. 安装 终端
3. 一个可以进行 root 操作的文件管理器（推荐 RE文件管理器 或者是 MT管理器）
4. 一双勤劳的手和会思考的大脑
当这些准备好之后你就可以开始了  



使用步骤：  

1. 下载外挂TWRP的压缩包，并解压到任意目录  
2. 在终端输入 su 获取 root  
3. 输入 sh 并输入空格，然后复制 rec 的绝对路径并粘贴在终端
4. 按照提示操作即可  
5. 直接进入 TWRP 时务必关闭输入法，否则……  
注：虽然说你可以直接按下HOME键，不过你需要确认终端可以在后台挂住
6. 一段时间的等待后，开始享受 外挂rec 的便利吧，如果出现问题请反馈
